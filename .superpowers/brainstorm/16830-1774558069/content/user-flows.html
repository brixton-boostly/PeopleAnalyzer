User Flows

Admin flow and Manager flow end-to-end

Admin Flow (Brixton)
Step 1
Create Review Cycle
Name it (e.g. "Q2 2025"), status = draft
→
Step 2
Sync from Gusto
Pull employees + manager relationships
→
Step 3
Review Assignments
Edit who reports to whom, confirm
→
Step 4
Activate Cycle
Status → active, managers can now submit
→
Step 5
Send Nudges
DM specific managers or all via Slack
→
Step 6
Monitor & Edit
Track completion, edit placements (audited)
Admin only All edits to review placements are recorded in the audit log with old → new values and a timestamp.
Manager Flow
Step 1
Receive Slack DM
Nudge with magic link to the app
→
Step 2
Auto-login
Token validated, session created
→
Step 3
Set Password
First login only — forced before continuing
→
Step 4
Place Employees
Click employee → click cell on 9-box grid
→
Step 5
Submit Review
Locked with timestamp, read-only after
→
Step 6
View History
Past cycles visible, read-only
Magic link Token is single-use and expires in 24h. After first use, manager can log in with email + password for subsequent visits.
